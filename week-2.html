Q10
Which of the following pair of prototypes is correct for overloading operator new and delete ?
a) void* operator new(size_t, char) and void operator delete(void *)
b) void* operator new(char, size_t) and void operator delete(void *)
c) void operator new(size_t, char) and void operator delete(void *)
d) void* operator new(size_t, char) and void operator delete(void)
Explaination

1. array is [10, 20, 30, 40].
2. so array[0] will give output 10.
3. If you know the pointer than you will understand it quickly.
4. variable array has base address of element 10.
5. so *array is same as array[0] and will give output 10.
6. Now compiler internally represent array[0] as *(array + 0) which will give output 10.
7. so 2[array-2] will be internally represent as *(2 + array -2) which will give output 10.
8. But our answer is -10.
9. At first time I am also confused but saw - (operator) and 2[array-2] as seperate things.
10. For example in printf("%d", -(a)); variable a and -(operator) are seperate expression and multiplied.
11. So -(operator) and value of 2[array-2] which is 10 is multiplied so answer is -10
