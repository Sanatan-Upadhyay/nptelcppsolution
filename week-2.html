Q6
a) 30, 30
b) 30, 30.2
c) 30.0, 30.2
d) Compilation error
Explanation

1. Function Add(a, b) will exact match to function Add(int a, int b).
2. For function Add(a, 20.2) compiler first match to Add(int a, int b) because 1st parameter exactly match and 2nd parameter can be converted to int to match.
3. Now for function Add(a, 20.2) compiler then match to Add(double a, double b) because 1st parameter can be converted to double to match and 2nd parameter exactly match.
4. So compiler confused and give ambigious error.
