Q3
a) 3 10
b) 3 4
c) 10 11
d) 4 10
Explanation

1. Inside main(), in cout fun() will be called.

2. Inside fun() i becomes 3 and an object is created. fun() will return post-increment value of i.

3. When the object obj gets out of scope, a destructor is called and i becomes 10.

4. Now, control come back to main() and gives output 3 and 10.

5. When you dry run ( with pen and paper) you will get output 3 10. But when you compile this program in codeblocks you will get output 3 0. If you print the value of i in next line you will get 3 10. Read about sequencing

Q4
Explanation

1. In class A,

a) Variable a is auto type which means every object will have seperate copy of variable a.
b) Variable b is static type which means a single copy of variable b will be shared by all the object.
c) A default constructor

2. Variable b is initialized outside the class to 0.

3. In main() three object are created. first is object x. Second is at position a[0]. Third is at position a[1].

4. For all three object variable a have different copy, 3 different memory allocation for variable a. Its means that all object have their own variable a and they can't see each other.

5. For all three object variable b have same copy, single memory allocation for variable b. Its means that all object have access to same variable b. If any changes made to the value of variable b then all three object will see this new value.

6. Object x calls the constructor. object x have variable a = 0 and variable n = 0. Inside constructor body variable n is incremented to n = 1 and after simple computation output is a = 1 n = 1 .

7. Object at a[0] calls the constructor and have variable a = 0 ( its own variable a initializes to 0 ) and variable n = 1 ( static variable other object have same variable b ) . Inside constructor body n is incremented to n = 2 and after simple computation output is a = 0 n = 2 .

8. Object at a[1] calls the constructor and have variable a = 0 ( its own variable a initializes to 0 ) and variable n = 2 ( static variable other object have same variable b ) . Inside constructor body variable n is incremented to n = 3 and after simple computation output is a = 3 n = 3 .

#include <string>
#include <iostream>
using namespace std;

class Test {
    string paper;
    int marks;
public:
    Test(): paper("Computer"), marks(0)      // Constructor 1
    { }

    Test(string p): paper(p), marks(0)       // Constructor 2
    { }

    Test(int m): paper("History"), marks(m)  // Constructor 3
    { }

    Test(string p, int m): paper(p), marks(m) // Constructor 4
    { }

    void Print() {
        cout << paper << " " << marks << endl;
    }
}; // End of Class definition 


int main() {

    string subject1, subject2;

    cin>> subject1 >> subject2;
    __________ // L1 using Constructor 1

    __________ // L2 using Constructor 2

    __________ // L3 using Constructor 3

   	__________ // L4 using Constructor 4

    t1.Print();
    t2.Print();
    t3.Print();
    t4.Print();

    return 0;
} // End main
Answer

Test t1; // L1

Test t2( subject1 ); // L2

Test t3( 50 ); // L3

Test t4( subject, 49 ); // L4
